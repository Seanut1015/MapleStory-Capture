# 使用說明
config.ini與Captue.exe務必放置於同資料夾中
1. 於GitHub下載ZIP並解壓縮
2. 取出其中dist資料夾(其他可以刪掉 不影響使用)
3. 執行Capture.exe
4. 選擇圖片儲存位置
5. 滑鼠移到楓之谷中的裝備上並按鍵盤"c鍵"(預設 可更改)
   
### 裝備窗請靠左
